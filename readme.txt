簡單店家 SSRM
================================================
SSRM stand for Simple Shop Resource Management.

介紹：
簡單店家是一個為了中小型服飾店家所打造的的一款線上ERP系統，目的是能讓店家透過簡單的操作，快速處理有關採購作業、進貨、退貨及庫存管理、結帳等等環節的資料處理工作。

功能：
- 採購作業：
-- 快速新增採購之商品名稱、成本、產品標價、規格（如顏色、尺寸）及採購件數。
-- 查看／新增／刪除／編輯歷史採購單。
-- 顯示採購單目前的進貨進度。
- 庫存作業：
-- 商品到貨後根據採購單新建進貨單開始進貨。
-- 根據採購單對已進貨之商品進行退貨作業。
-- 查看歷史進退貨單紀錄。
- 結帳作業：
-- 結帳功能
-- 顧客消費紀錄與金額累計
-- 當日營業數據呈現
-- 歷史結帳紀錄查看／刪除／編輯功能

技術：
- HTML, CSS and JavaScript
- SCSS for CSS
- React for JavaScript
- Rudux
- React Router
- SPA
- Firebase for Backend
- Facebook Login.
